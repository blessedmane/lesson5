GIT flow

  • GIT clone - для того, Чтоб склонить репозиторий
  • GIT pull - для того, чтоб получить изменения с репозитория
  • GIT checkout -b branchname - создать новую ветку и переключиться на нее
  • GIT add - добавить изменения, котор сделали локально
  • GIT commit -m "commit text" - подписать их
  • GIT push origin branchname - загрузить на гит сервер
